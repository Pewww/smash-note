# Smash-Note

## 게임 설명

알맞은 방향을 클릭하여 제한 시간 내에 많은 점수를 얻는 게임입니다.<br>
제한 시간은 기본적으로 **30초**가 주어지며, 특정 노트의 클릭을 통해 증가하거나 감소될 수 있습니다.
